## Design Principles

### **Visual Hierarchy**
1. **Primary**: Important actions (Add Expense, Save)
2. **Secondary**: Navigation, filters
3. **Tertiary**: Helper text, metadata

### **Color Usage**
**Normal Mode:**
- Primary: Blue (#3b82f6)
- Success: Green (#10b981)
- Warning: Yellow (#f59e0b)
- Error: Red (#ef4444)

**Halloween Mode:**
- Primary: Orange (#ff6b35)
- Success: Green (#22c55e)
- Warning: Gold (#ffd23f)
- Error: Red (#dc2626)

### **Spacing**
- Consistent 4px grid
- Generous whitespace
- Clear content grouping

### **Typography**
- Headers: Bold, larger
- Body: Regular, readable
- Metadata: Smaller, muted

### **Animations**
- Entrance: Fade + slide
- Exit: Fade out
- Hover: Scale + glow
- Loading: Shimmer/pulse
