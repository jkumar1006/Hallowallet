# Hackathon Context

## Project: HalloWallet
A Halloween-themed personal finance tracker built for a hackathon with innovative AI features.

## Key Differentiators

### 1. Voice-to-Expense Parsing
Natural language processing that converts casual speech into structured expense entries.
- "Spent 50 on groceries" → Parsed expense
- Multi-language support (6 Indian languages)
- Smart category detection

### 2. Smart Purchase Advisor
AI-powered financial advisor that analyzes purchase decisions in real-time.
- Budget impact analysis
- Savings goal tracking
- Alternative product suggestions
- Personalized recommendations

### 3. Halloween Theme
Unique spooky aesthetic with animations and interactive elements.
- Dark theme with orange/purple accents
- Animated ghosts, bats, pumpkins
- Glowing effects and transitions
- Toggle-able theme

### 4. Multi-language Support
Full internationalization for Indian users.
- English, Hindi, Kannada, Malayalam, Tamil, Telugu
- RTL support where needed
- Localized number formatting
- Cultural context awareness

## Hackathon Goals
- ✅ Innovative AI features
- ✅ Unique user experience
- ✅ Practical real-world application
- ✅ Technical complexity
- ✅ Polished presentation

## Demo Flow
1. **Sign Up/Login** - Quick authentication
2. **Voice Input** - "Spent 500 on dinner yesterday"
3. **Dashboard** - See parsed expense with charts
4. **Financial Profile** - Set income and savings goals
5. **Purchase Advisor** - "Should I buy this iPhone?"
6. **Smart Recommendation** - Get AI-powered advice
7. **Multi-language** - Switch to Hindi/Kannada

## Technical Highlights
- Next.js 14 with App Router
- TypeScript for type safety
- Custom NLP parser (no external APIs)
- JWT authentication
- Real-time calculations
- Responsive design
- Production-ready code
