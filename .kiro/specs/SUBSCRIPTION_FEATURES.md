## Technical Notes

### Database Schema
```typescript
type Expense = {
  id: string;
  userId: string;
  date: string;
  description: string;
  category: string;
  merchant?: string;
  amount: number;
  isSubscription?: boolean;
  notes?: string;
  // Future fields:
  subscriptionFrequency?: 'monthly' | 'yearly';
  nextRenewalDate?: string;
  autoGenerated?: boolean;
}
```

### API Endpoints Needed
- `GET /api/subscriptions` - List all subscriptions
- `POST /api/subscriptions/renew` - Manually trigger renewal
- `GET /api/subscriptions/upcoming` - Get upcoming renewals
- `POST /api/subscriptions/[id]/cancel` - Cancel subscription
