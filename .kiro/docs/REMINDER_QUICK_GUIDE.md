## Troubleshooting

**Q: Not seeing reminders?**
- Make sure you've set a reminder for that subscription
- Check if renewal date is within your reminder period
- Verify you haven't dismissed it recently

**Q: Too many popups?**
- Adjust reminder days to fewer days before renewal
- Remove reminders for subscriptions you don't need to track
- Dismiss popups you've already handled

**Q: Want to stop all reminders?**
- Go to each subscription and click "Remove"
- Or clear browser localStorage
